# PHP-Laravel
Repository For Laravel Projects

# Create Laravel Project
composer create-project laravel/laravel project-name

# Run Laravel Project
php artisan serve

# To generate a new application key:
php artisan key:generate

# Clear the configuration cache to ensure the changes take effect
php artisan config:clear

# If you are using any caching mechanisms, such as Redis or Memcached, clear those caches as well
php artisan cache:clear
